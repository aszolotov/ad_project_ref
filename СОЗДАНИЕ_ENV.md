# Создание файла .env

Файл `.env` с вашими настройками готов! 

## Быстрое создание

**Вариант 1: Через переименование**
1. Файл `.env.ready` уже создан с вашими настройками
2. Переименуйте его в `.env` (удалите `.ready`)

**Вариант 2: Через PowerShell**
```powershell
# Переименовать файл
Rename-Item -Path ".env.ready" -NewName ".env"
```

**Вариант 3: Создать вручную**
Создайте файл `.env` в корне проекта и скопируйте содержимое из `.env.ready`

## Важно!

После создания `.env` файла:
1. ✅ Проверьте, что файл `.env` в `.gitignore` (уже добавлен)
2. ✅ Убедитесь, что пароли не попадут в Git
3. ✅ Запустите приложение: `uvicorn backend.main:app --reload`
